# Chapter 4-2: 합성/집약 (Composition/Aggregation) - Has-a 관계

## 학습 목표

- 합성과 집약의 개념을 이해한다
- Has-a 관계와 Is-a 관계를 구분한다
- 상속의 오용을 피하고 합성을 선택하는 방법을 배운다

## 핵심 개념

### 1. Has-a 관계

"Car **has an** Engine" - 자동차는 엔진을 가진다.

### 2. 합성 vs 상속

상속 대신 합성을 선택해야 하는 경우가 많다.
