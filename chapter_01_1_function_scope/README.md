## 핵심 개념

### 1. 함수 정의

함수는 특정 작업을 수행하는 코드 블록이다.

```python
def 함수이름(매개변수1, 매개변수2):
    # 함수 본문
    return 반환값
```

### 2. 파이썬의 인자 전달 방식

파이썬은 **Call by object reference** 방식을 사용한다.

- **Immutable 객체** (int, str, tuple 등): 함수 내에서 값을 변경해도 원본이 변경되지 않는다
- **Mutable 객체** (list, dict, set 등): 함수 내에서 값을 변경하면 원본도 변경된다

이는 Call by value도 아니고 Call by reference도 아닌 파이썬만의 독특한 방식이다.

### 3. 스코프 (Scope)

변수가 접근 가능한 범위를 의미한다.

- **전역 스코프 (Global Scope)**: 함수 밖에서 선언된 변수
- **지역 스코프 (Local Scope)**: 함수 안에서 선언된 변수

지역 스코프에서는 전역 변수를 읽을 수 있지만, 수정하려면 `global` 키워드가 필요하다.

### 4. LEGB 규칙

파이썬은 변수를 찾을 때 다음 순서로 검색한다:

1. **L**ocal: 현재 함수 내부
2. **E**nclosing: 현재 함수를 감싸는 함수
3. **G**lobal: 모듈 최상위 레벨
4. **B**uilt-in: 파이썬 내장 함수/변수

## 주의사항

- 함수 내에서 전역 변수를 무분별하게 수정하는 것은 좋지 않은 습관이다
- 가능하면 함수의 매개변수로 필요한 값을 전달하고, 반환값으로 결과를 돌려주는 것이 좋다
- mutable 객체를 인자로 받을 때는 원본이 수정될 수 있음을 인지해야 한다
